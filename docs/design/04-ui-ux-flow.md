## 1. Design System

### 1.1 Color Palette

```
Primary (Blue Professional):
┌──────────────────────────────────────────────┐
│  #0D47A1  │  #1565C0  │  #42A5F5  │  #BBDEFB │
│  Dark     │  Primary  │  Light    │  Surface  │
└──────────────────────────────────────────────┘

Semantic Colors:
┌─────────────────────────────────────────┐
│  #4CAF50  │  #FFA726  │  #EF5350  │  #29B6F6 │
│  Success  │  Warning  │  Error    │  Info    │
└─────────────────────────────────────────┘

Neutral (Light mode):
┌──────────────────────────────────────────────────────┐
│  #FFFFFF  │  #F5F5F5  │  #E0E0E0  │  #757575  │  #212121 │
│  Bg       │  Surface  │  Border   │  Text-2nd │  Text-1st│
└──────────────────────────────────────────────────────┘

Neutral (Dark mode):
┌──────────────────────────────────────────────────────┐
│  #121212  │  #1E1E1E  │  #2C2C2C  │  #B0B0B0  │  #FFFFFF │
│  Bg       │  Surface  │  Border   │  Text-2nd │  Text-1st│
```

### 1.2 Typography

```
Font Family:
├── Thai: Noto Sans Thai
├── Lao:  Noto Sans Lao
└── Latin: Inter

Scale:
├── H1:  24sp  Bold    (หัวข้อหลัก)
├── H2:  20sp  SemiBold (หัวข้อรอง)
├── H3:  18sp  SemiBold (หัวข้อย่อย)
├── Body: 16sp Regular  (เนื้อหา)
├── Body2: 14sp Regular (เนื้อหารอง)
├── Caption: 12sp Regular (คำอธิบาย)
└── Price: 24sp Bold Tabular (ตัวเลขราคา — ตัวเลขกว้างเท่ากัน)
```

### 1.3 Spacing & Grid

```
Base unit: 4dp
Spacing scale: 4, 8, 12, 16, 20, 24, 32, 40, 48, 64

Content padding:
├── Phone:  16dp
├── Tablet: 24dp
└── Desktop: 32dp

Card radius: 12dp
Button radius: 8dp
Input radius: 8dp
```

### 1.4 Icon System

```
Icon set: Material Symbols (Rounded)
Size:
├── Small: 20dp (ใน list, badge)
├── Default: 24dp (ทั่วไป)
├── Large: 32dp (หน้า POS, ปุ่มลัด)
└── XLarge: 48dp (empty state, onboarding)
```

### 1.5 Component Library

```
Buttons:
├── PrimaryButton — สีน้ำเงิน filled (ชำระเงิน, ยืนยัน)
├── SecondaryButton — outlined (ยกเลิก, รอง)
├── DangerButton — สีแดง (Void, ลบ)
├── IconButton — แค่ icon (edit, delete, print)
├── QuickKeyButton — ปุ่มสินค้าลัด (ใหญ่, มีรูป+ชื่อ+ราคา)
└── NumpadButton — ปุ่มตัวเลข (สำหรับ touch)

Cards:
├── ProductCard — รูป + ชื่อ + ราคา (grid)
├── ProductRow — icon + ชื่อ + ราคา + qty (list)
├── CartItemCard — ชื่อ + qty + ราคา + modifiers + note
├── OrderCard — เลขบิล + สถานะ + ยอดรวม + เวลา
├── TableCard — ชื่อโต๊ะ + สถานะ + เวลา + ยอด
├── KitchenCard — เลขบิล + โต๊ะ + รายการ + timer
└── MemberCard — ชื่อ + เบอร์ + ยอดรวม

Dialogs:
├── ConfirmDialog — ยืนยัน + ยกเลิก
├── ModifierDialog — เลือก modifier + note
├── DiscountDialog — ใส่ส่วนลด
├── VoidReasonDialog — ใส่เหตุผล
├── PinDialog — ใส่ PIN (ยืนยันสิทธิ์)
└── NoteDialog — พิมพ์โน้ต

Inputs:
├── SearchField — พร้อม icon scan barcode
├── NumpadInput — กดตัวเลขจาก numpad ในแอป
├── CurrencyInput — ใส่จำนวนเงิน + สกุลเงิน
└── QuantitySelector — ปุ่ม +/- หรือพิมพ์ตรง
```

### 3.3 Table Flow

```
Floor Plan Screen
    │
    ├── เห็นโต๊ะทุกตัว (สี = สถานะ)
    │   🟢 ว่าง  🔴 มีลูกค้า  🟠 จอง  🟣 รวมโต๊ะ
    │
    ├── กดโต๊ะว่าง → สร้าง Order ใหม่ → ไป POS
    │
    ├── กดโต๊ะมีลูกค้า → เปิด Order เดิม → ไป POS
    │
    └── Long press / Action Menu:
        ├── ย้ายโต๊ะ (Move) → เลือกโต๊ะปลายทาง
        ├── รวมโต๊ะ (Merge) → เลือกโต๊ะที่จะรวม
        ├── แยกโต๊ะ (Split) → เลือกรายการที่จะแยก
        └── ดูรายละเอียดออเดอร์
```

### 3.4 Kitchen Flow (KDS)

```
Kitchen Display Screen
    │
    ├── แสดง Card เรียงตามเวลาที่สั่ง (FIFO)
    │   ┌─────────────────────────────────┐
    │   │  🔴 NEW   │ 🟠 PREPARING │ 🟢 READY │
    │   │  (ใหม่)    │ (กำลังทำ)     │ (เสร็จ)   │
    │   └─────────────────────────────────┘
    │
    │   แต่ละ Card แสดง:
    │   ├── เลขบิล + ชื่อโต๊ะ
    │   ├── รายการอาหาร + จำนวน
    │   ├── Modifier + Note
    │   ├── ⏱ เวลาที่ผ่านไป (timer สี)
    │   │   ├── < 5 นาที → เขียว
    │   │   ├── 5-10 นาที → เหลือง
    │   │   └── > 10 นาที → แดง (กระพริบ)
    │   └── ปุ่ม: Accept → Ready → Served
    │
    └── เสียง 🔔 เมื่อมี order ใหม่เข้า
```

## 7. Accessibility

### 7.1 Requirements

| Area | Requirement |
|------|------------|
| Touch Target | ≥ 48×48dp ทุกปุ่ม (POS ใช้ ≥ 56×56dp) |
| Contrast | ≥ 4.5:1 (text), ≥ 3:1 (large text/icons) |

| Color Blindness | ไม่ใช้สีเป็นตัวแยกแยะอย่างเดียว (เพิ่ม icon/text) |

| Keyboard | Tab navigation สำหรับ Desktop/POS machine |
| Focus Indicator | Visible focus ring สำหรับ keyboard navigation |
